### Exemples courants d’appliances réseau

- **Pare-feu / UTM** : Fortinet, Palo Alto Networks  
- **VPN** : OpenVPN  
- **Proxy / filtrage web** : Squid  
- **IDS / IPS** : Suricata  

### Formes de déploiement possibles

Ces solutions existent souvent :
- en **boîtier physique**,
- en **machine virtuelle**,
- en **conteneur**.
